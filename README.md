# Avalonia Kanban Demo

A lightweight Kanban board application built as a demo/test project using [Avalonia](https://avaloniaui.net/), a cross-platform .NET UI framework. Focus on drag-and-drop interactions and MVVM architecture.

Prerequisite: .NET 9.0 or newer
Uses: Avalonia 11.3.11

Run the application with:

```bash
dotnet run
```
